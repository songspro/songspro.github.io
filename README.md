# SongPro Documentation

This repository contains the documentation website for the SongPro format, built with Hugo and deployed to GitHub Pages.

## About SongPro

SongPro is a text file format for making chord & lyric lead sheets for songs.

## Development

This site is built using [Hugo](https://gohugo.io/), a fast and modern static site generator.

### Local Development

To run the site locally, you need to have Hugo installed. Then run:

```bash
hugo server
```

The site will be available at `http://localhost:1313/`.

## Deployment

The site is automatically built and deployed to GitHub Pages using GitHub Actions whenever changes are pushed to the `main` branch.
